<h1>sdkMicrophoneCS_UploadToBlob</h1>
<p>This repository contains a sample that shows how to extend the Windows Phone sdkMicrophoneCS sample to upload recorded sounds into Windows Azure blob storage.
</p>

<h3>Dependencies</h3>

</ul>
